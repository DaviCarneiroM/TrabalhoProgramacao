Projeto sinaleiro

Projeto sinaleiro é o nome dado ao primeiro projeto desenvolvido na disciplina de robótica, sua finalidade tem em simular um sinaleiro a partir das plataformas disponibilizadas pelo estado e depois refazer usar os objetos disponibilizados pelo colégio.

Para melhor entendimento, você pode pode receber mais detalhes aqui.

Tinkercad

Tinkercad é uma plataforma educativa usada para projetar protótipos de robótica.

Para iniciarmos o projeto sinaleiro, usamos essa plataforma e simulamos ele começando pelo básico, ou seja, a gente fez o sinaleiro normal primeiro.

Você pode analisar ele pela imagem à direita.

Sinaleiro pedestre

O projeto foi além, e depois tinhamos que criar um sinaleiro para pedestre.

Para isso, foi adicionado mais dois leds como visto na imagem, e foi mudado algumas coisas no circuito e no código do sinaleiro, assim, metafóricamente, ajudando os pedestres a passarem.

Você pode analisar pela imagem à direita.
