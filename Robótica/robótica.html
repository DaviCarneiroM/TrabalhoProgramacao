Projeto sinaleiro

Projeto sinaleiro é o nome dado ao primeiro projeto desenvolvido na disciplina de Robótica, seu objetivo é construir um sistema que se assemelhe a uma sinaleiro através do computador com as plataformas disponibilizadas pelo estado, e depois refazer o projeto usando os objetos fornecidos pelo colégio.

Material e métodos:

Arduino Uno, placa de ensaio pequena, resistores(3), leds/pinos(3), fios(7) e cabo com entrada USB. Para iniciar a montagem do circuito, é preciso primeiro conectar o Arduino na placa de ensaio pequena utilizando um fio que saia do GND do Arduino e vá até a coluna do cátodo na placa. Em seguida, vamos colocar os LEDs nas colunas G, H e I, deixando um espaço de duas ou mais casas entre eles. Para criar um sinalizador, vamos trocar as cores dos LEDs respectivamente para verde, amarelo e vermelho. Agora, vamos conectar os fios na parte negativa de cada LED, que deve ser ligada à coluna do cátodo na placa. Em seguida, vamos conectar os três resistores de 220 ohms na parte positiva de cada LED. Para finalizar a montagem, vamos conectar cada resistor à placa, que possui casas numeradas de 0 a 13.

Código em blocos:

No código, vamos começar entendendo-o em blocos. Primeiramente, usaremos o bloco "para sempre", que deixará o nosso código em loop. Em seguida, vamos definir o primeiro LED (pino), que será o verde, como ligado, utilizando o bloco "definir pino [número da casa no Arduino] como ALTO" dentro do "para sempre". Logo após, colocaremos o bloco "aguardar [7] [S (segundos)]" e, abaixo dele, outro bloco de definir pino, com o mesmo número de casa, mas com a descrição "BAIXO", que deixará o LED desligado. Repetiremos o processo para o LED amarelo, mudando o número da casa e a temporização para três segundos.Para o LED vermelho, repetiremos o processo de temporização, deixando-o ligado por cinco segundos.

Código em texto:

No código em texto, usaremos o comando "void setup()", abrindo chaves nele, para dizer que os componentes citados devem iniciar junto ao programa. Dentro dele, utilizaremos três comandos "pinMode([número da casa no Arduino], OUTPUT)", configurando os LEDs verde, amarelo e vermelho. Fecharemos as chaves do "void setup()" e escreveremos o comando "void loop()" abrindo chaves, para que os componentes citados entrem em loop. Dentro dele, utilizaremos o código "digitalWrite([número da casa no Arduino], HIGH)" para o LED verde, seguido do comando "delay(7000)" para manter o LED aceso por 7 segundos e, por último, o comando "digitalWrite([número da casa no Arduino], LOW)" para desligá-lo. Repetiremos esse processo para o LED amarelo, mas com um delay de 3000 milissegundos e para o LED vermelho, com um delay de 5000 milissegundos. É importante lembrar que o Tinkercad utiliza da linguagem C++.
