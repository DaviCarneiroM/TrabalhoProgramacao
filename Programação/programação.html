Este é um site dedicado a reúnir as pesquisas realizadas na disciplina de Programação.

A primeira pesquisa que realizamos, foi sobre as principais tags da linguagem de programação HTML, onde você pode ler mais abaixo.

Principais tags HTML

1. `<html>`: Define o início e o fim do documento HTML.
2. `<head>`: Contém informações sobre o documento, como título, estilo CSS e metadados.
3. `<title>`: Define o título do documento que é exibido na barra de título do navegador.
4. `<body>`: Contém o conteúdo visível do documento, como texto, imagens, links e elementos interativos.
5. `<h1>` a `<h6>`: Define títulos e subtítulos, sendo `<h1>` o mais importante e `<h6>` o menos importante.
6. `<p>`: Define um parágrafo de texto.
7. `<a>`: Cria um link para outra página ou recurso.
8. `<img>`: Insere uma imagem no documento.
9. `<ul>` e `<li>`: Criam listas não ordenadas, onde cada item é marcado com um ponto.
10. `<div>`: Define uma divisão ou seção no documento para agrupar elementos.
11. `<form>`, `<input>`, `<button>`: Criam um formulário para envio de dados, com campos de entrada e botões.
12. `<header>`, `<nav>`, `<footer>`, `<section>`: São tags semânticas que definem partes específicas da página, como cabeçalho, navegação, rodapé e seções.
